공부한 내용들 : 

→ h${hello world}*6 적고 탭 누르면, "hello wrold" 의 내용이 h1~h6 까지 생성된다.

(h(헤딩)은 h1에서 h6 까지만 존재한다, 문서 내에는 h1이 하나만 있는것을 추천한다.)

→ p 태그는, 문단을 나타낸다. <p></p>

→ 5sub2 는, 5<sub>2</sub>

→ 2sup4 는, 2<sup>4</sup>

→ Lorem : 의미없는 문구
→ br 로 줄 이동, 안 닫아도 됨 <br>
